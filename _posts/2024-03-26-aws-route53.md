## 1장. GitHub Action이란

### 다양한 CI/CD 도구들 중 왜 Github Action인가?

* 먼저 Github Action이란, GitHub에서 공식적으로 제공하는 CI/CD툴입니다.
* 리포지토리(저장소)에서 바로 소프트웨어 개발 워크플로를 자동화, 사용자 지정 및 실행합니다.
